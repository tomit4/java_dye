# Introduction to Java Programming: Final Touchstone Project

This repository contains my final project for
[Sophia.org](https://www.sophia.org/)'s _Introduction to Java Programming_
course. The project is a command-line tool called **Dye**, which allows users to
convert colors between **RGB**, **HEX**, and **HSL** color models.

## Features

- Convert between RGB, HEX, and HSL color models.
- Input validation ensures values are within valid ranges.
- Provides a help message and allows quitting at any time.
- Continuous operation: users can perform multiple conversions without
  restarting the program.

## Getting Started

To run the program locally:

1. Clone the repository:

```bash
git clone https://github.com/tomit4/java_dye
```

2. Navigate into the project directory:

```bash
cd java_dye
```

3. Compile all Java files:

```bash
javac ./.*.java
```

4. Run the main program:

```bash
java Dye
```

Follow the on-screen instructions to use the color conversion tool.
